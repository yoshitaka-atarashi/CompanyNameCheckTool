# 使用ガイド

PowerPoint Keyword Detection Tool の使用方法を説明します。

## 基本的な使用方法

### ステップ 1: アプリケーションの起動

```bash
python app.py
```

ブラウザで `http://localhost:5000` にアクセスします。

### ステップ 2: PowerPoint ファイルのアップロード

1. ページの「ファイルを選択」ボタンをクリック
2. PowerPoint ファイル（`.pptx` または `.ppt`）を選択
3. ファイルがアップロードされます

### ステップ 3: キーワードの入力

検出、削除、または置換したいキーワードを入力フィールドに入力します。

複数のキーワードを検出する場合は、以下のいずれかの方法で入力できます：

- **1 つずつ入力**: 「追加」ボタンで複数のキーワードを追加
- **一度に複数入力**: カンマで区切ったキーワードを入力

### ステップ 4: 操作の選択と実行

以下の 3 つの操作から選択します：

#### 検出（Detect）
- PowerPoint 内のキーワードを検出し、位置と出現回数を表示します
- ファイルは修正されません

```json
{
  "results": [
    {
      "slide": 1,
      "shape": 0,
      "text": "Company XYZ",
      "keywords": ["Company"],
      "count": 1
    }
  ]
}
```

#### 削除（Delete）
- 指定したキーワードをファイルから削除します
- 修正済みファイルがダウンロードされます

#### 置換（Replace）
- キーワードを別の文字列に置換します
- 置換後の文字列を指定してください
- 修正済みファイルがダウンロードされます

### ステップ 5: ファイルのダウンロード

操作が完了すると、修正済みのファイルが自動的にダウンロードされます。

ダウンロードされたファイル名は以下のフォーマットです：

- 検出：`original_detect.pptx`
- 削除：`original_deleted.pptx`
- 置換：`original_replaced.pptx`

## 使用例

### 例 1: 会社名の検出

1. PowerPoint ファイルをアップロード
2. キーワード: `Company`
3. 操作: **検出**
4. 結果: 会社名が含まれるスライドと位置が表示されます

### 例 2: 社外秘情報の削除

1. PowerPoint ファイルをアップロード
2. キーワード: `機密`, `社外秘`
3. 操作: **削除**
4. 修正済みファイルをダウンロード

### 例 3: テンプレートの置換

1. PowerPoint ファイルをアップロード
2. 検出キーワード: `[Company]`
3. 置換先: `Acme Corp`
4. 操作: **置換**
5. 修正済みファイルをダウンロード

## 制限事項

- **ファイルサイズ**: 最大 50MB
- **対応形式**: `.pptx`, `.ppt`
- **テキスト検出**: スライドのテキストのみ検出します（画像内のテキストは対象外）
- **複雑な書式**: 一部の複雑な書式は保持されない場合があります

## よくある質問

### Q: 複数のファイルを同時に処理できますか？
A: 現在は 1 つのファイルずつ処理します。複数のファイルを処理する場合は、繰り返し実行してください。

### Q: 大文字と小文字は区別されますか？
A: 大文字と小文字は区別されません。`Company` と `company` は同じものとして扱われます。

### Q: 部分一致で検出できますか？
A: はい。例えば、`com` を入力すると `company` も検出されます。

### Q: 元のファイルは保持されますか？
A: はい。操作結果は新しいファイルとしてダウンロードされ、元のファイルは変更されません。
