# 変更ログ

## [1.0.0] - 2025-11-19

- PowerPoint ファイル内のキーワード検出機能

- キーワード置換機能
- Web UI でのファイルアップロード
- 修正済みファイルのダウンロード機能
- PowerPoint ファイルの診断ツール

### 技術スタック
- Flask 2.3.3
- python-pptx 0.6.23
- Python 3.8以上
